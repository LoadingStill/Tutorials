# Simple Budget Application

The Simple Budget Application is a Python-based budgeting tool that allows users to track their income, expenses, and savings. It provides a simple and user-friendly interface to input financial data and calculate the savings based on the income and expenses.

## Features

- Track income and expenses
- Calculate savings
- Save and open budget files
- Support for multiple months of budget data
- Error handling for input validation

## Requirements

- Python 3.x
- Tkinter library

## Installation

1. Clone the repository or download the source code.
2. Ensure you have Python 3.x installed on your system.
3. Install the required dependencies by running the following command:


pip install tkinter  
  
## Usage

1. Run the `budget_app.py` file using Python.
2. Enter your income and expenses in the respective fields.
3. Click the "Calculate" button to calculate your savings.
4. Use the "Save" button to save your budget data to a file.
5. Use the "Open" button to open a previously saved budget file.
6. The application supports multiple months of budget data. Each month's data can be saved as a separate file and opened independently.
